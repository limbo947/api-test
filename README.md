# OpenAI API 调试器

一个纯前端、无框架依赖的 OpenAI 兼容 API 调试工具，支持 Chat Completions 与 Responses 接口，为开发者提供安全高效的 API 测试体验。

## 核心功能

- **双模式编辑**：可视化表单模式配置请求参数，或切换至 JSON 模式自由编辑原始请求体，两种模式实时同步
- **对话式交互**：内置聊天界面，支持多轮对话、流式输出、系统提示词设置、模型选择与历史会话管理
- **多模态文件上传**：支持图片、音频、视频上传，自动转为 base64 嵌入请求体
- **流式响应渲染**：SSE 流式输出增量更新，避免频繁 DOM 重建，支持思考链（reasoning）内容展示
- **多格式响应预览**：自动识别并展示 JSON、图片、音频、视频等响应类型
- **智能重试机制**：遇到 429/502/503/504 错误时自动重试，采用指数退避策略
- **配置方案管理**：保存/加载/导入/导出多套配置方案，敏感信息存储时自动混淆
- **密钥管理**：独立管理多个 API Key，支持按提供商分类，存储时混淆保护
- **历史记录**：自动保存最近请求记录，支持一键还原与导出，脱敏处理
- **分享与导入**：通过 URL 分享当前配置（敏感信息自动过滤），支持加密链接
- **主题切换**：内置翡翠科技、琥珀工坊、星际幽蓝三套深色主题
- **模型获取**：自动从 `/models` 端点拉取可用模型列表
- **自定义请求头**：支持任意 HTTP 方法与自定义请求头配置

## 项目结构

```
web/
├── index.html    # 页面结构与 CSP 安全策略
├── style.css     # 暗色主题样式，响应式布局
├── config.js     # 默认配置、端点模板、工具函数、密钥管理、缓存系统
├── app.js        # 主应用逻辑，模块化架构
├── theme.js      # 主题切换功能
├── upload.js     # 文件上传与多模态处理
└── share.js      # URL 分享与加密
```

## 快速开始

直接在浏览器中打开 `web/index.html`，或在本地启动静态服务：

```bash
# Node.js
npx serve web

# Python
python -m http.server -d web 8080
```

打开后，配置 Base URL、端点路径与 API Key，即可发送请求并查看响应。

## 技术亮点

- **零依赖**：纯原生 JavaScript，使用 IIFE 模块化设计，无框架依赖
- **安全优先**：CSP 策略、敏感信息混淆存储、URL 分享自动脱敏、安全的 DOM API 操作
- **性能优化**：SSE 流式增量渲染、输入防抖同步、LRU 请求缓存与请求去重
- **健壮性**：localStorage 降级处理、`structuredClone` 优先 + JSON 回退的深拷贝、完整的配置校验机制
- **可访问性**：键盘导航支持、ARIA 属性、跳过导航链接、减少动画偏好适配
- **响应式设计**：适配桌面、平板与移动端三种屏幕尺寸

## 兼容性

支持所有现代浏览器（Chrome 90+、Firefox 90+、Safari 15+、Edge 90+）。依赖浏览器原生 Fetch API、Web Crypto API 与 localStorage。
